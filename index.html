Introducción

La API de Books (interfaz de programación de aplicaciones) le permite acceder a una base de datos de libros falsos de prueba. Puede utilizar la API para crear sus propios sitios web o aplicaciones dinámicos respaldados por los mismos datos que se generan como datos falsos atraves de factories y datos semillas en laravel.

Con la API de Books puede:


Buscar mediante el id y muestre la información del libro especifico. Ademas consulte listado completo de libros. Ademas le permite agregar sus propios libros, eliminarlos y actualizarlos.

Esta es una API RESTful, lo que significa que utiliza URL predecibles para acceder a los recursos y, en caso de error, devuelve códigos de respuesta HTTP significativos. Como es el caso del error 404, al consultar el id de un libro no existente. Esto permite el uso de autenticación GET, POST y HTTP, que los clientes HTTP estándar entienden.
